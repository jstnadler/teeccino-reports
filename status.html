Pipeline Scripts

1 ingest_shopify.py
Shopify GraphQL Admin API → raw_orders, raw_line_items, raw_discount_allocations, order_journey, order_moments
2 ingest_meta.py
Meta Marketing API → raw_meta_ads + raw_meta_ad_insights. Daily account-level + per-ad monthly insights with thumbnails.
3 ingest_google.py
Google Ads CSV → raw_google_ads. API not implemented — manual CSV import only.
4 ingest_shipstation.py
ShipStation REST API → raw_shipments. Supports --days N, --month, and full backfill.
5 transform_daily.py
Aggregates raw data → calc_daily_shopify. DTC/wholesale splits, True Gross, discount breakdown, subscriptions.
6 transform_sku.py
Line items → calc_daily_sku_sales. Per-SKU daily units, gross, net, COGS by channel.
7 transform_attribution.py
Journey data → calc_attribution. First/last touch source, medium, campaign, attributed channel.
8 build_mtd_dashboard.py
MTD KPI HTML with YoY, daily trends, discount waterfall, top ads. Pushes to GitHub Pages.
9 build_ytd_dashboard.py
YTD KPI HTML with monthly breakdowns and cumulative pacing. Pushes to GitHub Pages.
10 build_all_matrices.py
Sales & Revenue (4 sheets), Product Performance (7 sheets), Advertising & Attribution (3 sheets). Excel to OneDrive.
!

Known Issues

  • WARNING Google Ads OAuth token can expire — If Google data stops updating, run python google_ads_reauth.py to re-authorize.
  • WARNING Meta API token expires ~every 60 days — If Meta data stops updating, the access token needs a manual refresh.
  • WARNING Excel file locks — Matrix builds fail with PermissionError if workbooks are open in Excel/OneDrive. Non-critical.
  • INFO ShipStation rate limit — 40 req/min. Large backfills take 15-20 minutes with throttling.
  • INFO Forecast file hardcoded — 2026 Forecast v3.xlsx referenced directly. Update manually when new file is created.
