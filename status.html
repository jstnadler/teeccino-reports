Orchestrator

Single entry point: python daily_run.py — detects missing dates, runs all stages in sequence, and performs data quality checks. Individual scripts can also be run standalone for targeted operations.

S

Pipeline Scripts

1 ingest_shopify.py
Fetches orders from Shopify GraphQL Admin API. Populates raw_orders, raw_line_items, raw_discount_allocations, order_journey, and order_moments. Supports single-date, date-range, and backfill modes.
2 ingest_meta.py
Fetches account-level daily data and per-ad monthly insights from Meta Marketing API. Stores in raw_meta_ads and raw_meta_ad_insights. Downloads ad thumbnails locally. Supports --api, --ads, and backfill-ads modes.
3 ingest_google.py
Loads Google Ads data into raw_google_ads. Currently CSV-import only — the --api flag prints a warning. Needs API implementation or regular CSV export.
4 ingest_shipstation.py
Fetches shipment data from the ShipStation REST API. Populates raw_shipments. Supports --days N, --month YYYY-MM, and full backfill modes. Rate-limited to 40 req/min.
5 transform_daily.py
Aggregates raw order data into calc_daily_shopify — the core KPI table. Computes DTC/wholesale splits, True Gross, discount breakdowns, subscription metrics, and free item tracking.
6 transform_sku.py
Aggregates line items into calc_daily_sku_sales with per-SKU daily units, gross, net, discounts, and COGS. Separates DTC, wholesale, and subscription channels.
7 transform_attribution.py
Processes order_journey touchpoints into calc_attribution with first-touch/last-touch source, medium, campaign, and attributed channel classification.
8 build_mtd_dashboard.py
Generates MTD KPI HTML dashboards with YoY comparisons, daily trend charts, discount waterfall, top ads cards, and forecast pacing. Pushes to GitHub Pages.
9 build_ytd_dashboard.py
Generates YTD KPI HTML dashboards with monthly breakdowns, cumulative trends, and year-over-year comparisons. Pushes to GitHub Pages.
10 build_all_matrices.py
Builds Excel workbooks: Sales & Revenue Matrix (4 sheets), Product Performance (7 sheets), and Advertising & Attribution (3 sheets). Output to OneDrive.

Known Issues

  • BLOCKED Google Ads API not implemented — ingest_google.py --api prints a warning and exits. Data comes from manual CSV import only. Last import: 2026-04-01.
  • WARNING Meta API token expires ~every 60 days — If Meta data stops updating, the access token needs a manual refresh. Check ingest_meta.py logs for auth errors.
  • WARNING Excel file locks — Matrix builds fail with PermissionError if workbooks are open in Excel/OneDrive. Logged as non-critical.
  • INFO ShipStation rate limit — 40 requests per minute. Normal daily runs are fine; large backfills (28+ months) take 15-20 minutes with throttling.
  • INFO Forecast file is hardcoded — 2026 Forecast v3.xlsx is referenced directly. Needs manual update when a new forecast file is created.
  • INFO Windows encoding — Console output uses ASCII only. Set PYTHONIOENCODING=utf-8 if running manually from terminal.
