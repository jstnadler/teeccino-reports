📜 Monthly Price List Generator (manual)
C:\Users\JustinAdler\clawd\Automations\generate_monthly_price_reference.py
ACTIVE
HOW PRICE LISTS ARE BUILT:
1. Fetches ALL orders for a given month from Shopify GraphQL.
2. Excludes subscription orders and wholesale orders — only uses one-time DTC purchases.
3. For each SKU, collects every originalUnitPrice observed across all qualifying orders.
4. Calculates the reference price as the mode (most common price). Falls back to median if no clear mode.
5. Saves as PriceList/Monthly/prices_YYYY-MM.json — one file per month.

Coverage: Jan 2024 → Feb 2026 (26 months).
⚠️ Not automated daily. Run manually when new months need prices or when price changes happen.
📥 Inputs: Shopify GraphQL API (historical order line items)

🗄️ Analytics DB Pipeline (7:15 AM daily)
C:\Users\JustinAdler\clawd\Automations\analytics\
ACTIVE
The core data pipeline. Ingests raw data directly from APIs into a SQLite analytics database, then runs transforms to calculate daily aggregates, SKU-level sales, attribution, and customer cohorts. All calculations use True Gross (Units × Price List).

Pipeline steps (in order):
1. ingest_shopify.py — Fetches orders, line items, discount allocations, customer journey moments from Shopify GraphQL
2. ingest_meta.py --api — Fetches daily Meta Ads spend, impressions, clicks, conversions directly from Meta API
3. ingest_google.py --api — Fetches daily Google Ads spend, impressions, clicks, conversions directly from Google Ads API
4. transform_daily.py — Aggregates into calc_daily_shopify: DTC/WS orders, True Gross, revenue, discount components, free items, sub tracking
5. transform_sku.py — Aggregates into calc_daily_sku_sales: per-SKU True Gross, net, discounts, DTC/WS/sub splits
6. transform_attribution.py — Calculates calc_attribution: NC/RC classification, channel attribution from customer journey moments
7. transform_cohorts.py — Calculates calc_customer_cohorts: cohort month, LTV windows (30d/90d/365d), order frequency

Key design decisions:
• All API ingestion is direct — no CSV intermediary
• True Gross = Units × Price List price (falls back to nearest prior month if current month missing)
• True Discounts = True Gross − Net (subtotal), NOT Shopify's total_discounts
• DTC-only filtering: is_wholesale = 0 on all DTC metrics
• Ignores non-physical SKUs: DONATE, GIFTCARD, SP001, GUIDEPROTECTION-*, EDONATION-*
• ~148K orders, ~505K line items, ~265K customer journey moments
📥 Inputs: Shopify GraphQL API, Meta Ads API, Google Ads API, PriceList/Monthly/*.json, COGS/Cogs.xlsx
