🔄 How the Database Gets Updated

The analytics pipeline runs daily at 7:00 AM PST via run_pipeline.py. It can also be triggered manually for any date or as a full rebuild.

── Step 1: Ingestion ──
ingest_shopify.py → Shopify GraphQL Admin API → raw_orders + raw_line_items + raw_discount_allocations + order_journey + order_moments
ingest_meta.py → Daily Meta Ads CSV (or Meta API) → raw_meta_ads
ingest_google.py → Daily Google Ads CSV (or Google API) → raw_google_ads

── Step 2: Transform ──
transform_daily.py → raw_orders + raw_line_items → calc_daily_shopify (37 daily aggregate columns)
transform_sku.py → raw_line_items + price_list + cogs → calc_daily_sku_sales (per-SKU daily metrics)
transform_attribution.py → raw_orders + order_journey → calc_attribution (first/last touch per order)
transform_cohorts.py → raw_orders → calc_customer_cohorts (full rebuild only — too expensive daily)

── Step 3: Output ──
build_mtd_dashboard.py → calc_daily_shopify + raw_meta/google_ads → mtd.html → GitHub Pages
Update Schedule: The daily pipeline runs at 7:00 AM PST, processing the previous day's data. A full rebuild (all dates since Jan 2023) takes ~15 minutes.

Commands:
python run_pipeline.py — Transform today's data
python run_pipeline.py --ingest — Ingest from APIs + transform
python run_pipeline.py --date 2026-04-01 — Process a specific date
python run_pipeline.py --full — Full rebuild of all calculated tables
