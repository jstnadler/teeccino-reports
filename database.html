How the Database Gets Updated

The analytics pipeline runs daily at 6:00 AM via daily_run.py (Windows Task Scheduler). It can also be triggered manually for any date or as a full rebuild.

— Step 1: Ingestion —
ingest_shopify.py → Shopify GraphQL Admin API → raw_orders + raw_line_items + raw_discount_allocations + order_journey + order_moments
ingest_meta.py → Meta Marketing API → raw_meta_ads + raw_meta_ad_insights + raw_meta_ads_campaigns
ingest_google.py → Google Ads API → raw_google_ads + raw_google_ads_campaigns
ingest_shipstation.py → ShipStation REST API → raw_shipments

— Step 2: Transform —
transform_daily.py → raw_orders + raw_line_items → calc_daily_shopify (43 daily aggregate columns)
transform_sku.py → raw_line_items + price_list + cogs → calc_daily_sku_sales (per-SKU daily metrics)
transform_attribution.py → raw_orders + order_journey → calc_attribution (first/last touch per order)
transform_cohorts.py → raw_orders → calc_customer_cohorts (full rebuild only — too expensive daily)

— Step 3: Output —
build_all_matrices.py → calc tables + raw tables → 3 Excel workbooks (14 sheets)
build_mtd_dashboard.py → calc_daily_shopify + raw_meta/google_ads + raw_meta_ad_insights → mtd.html → GitHub Pages
build_ytd_dashboard.py → calc_daily_shopify + raw_meta/google_ads → ytd.html → GitHub Pages
build_ops_dashboard.py → raw_shipments → ops.html → GitHub Pages
Update Schedule: The daily pipeline runs at 6:00 AM via Windows Task Scheduler (TeeccinoDataOps-DailyRun), processing the previous day's data. A full rebuild (all dates since Jan 2023) takes ~15 minutes.

Commands:
python daily_run.py — Full daily pipeline (ingest + transform + build)
python ingest_shopify.py 2026-04-01 — Ingest a specific date
python transform_daily.py 2026-04-01 2026-04-06 — Transform a date range
python build_mtd_dashboard.py --all — Rebuild all MTD dashboards
