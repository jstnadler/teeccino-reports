Schema Reference

Table: raw_upsell_attributions One row per accepted upsell line item. Source: Shopify GraphQL Orders API, line_items[].customAttributes with keys prefixed _upsell_*. Underscore-prefix is Shopify convention for hidden line-item properties — invisible to customer-facing emails/checkout but exportable via the API.
Source attribute keys (set by ThemeSmith's modal controllers) _upsell_modal_v · _upsell_source_product_id · _upsell_source_variant_id · _upsell_campaign · _upsell_source_mode · _upsell_discount_pct · _upsell_accepted_at
